# Table: QUOTES (sorted by File)
```dataview
TABLE WITHOUT ID
	file.link AS "File*",
	level AS "Level",
	source AS "Book",
	ptopic AS "Topic",
	ptopic.level AS "Topic Level"
FROM "content/QUOTES"
SORT title ASCENDING
```
